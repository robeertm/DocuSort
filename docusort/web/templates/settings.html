First-run hints (Gatekeeper, dependencies)
  • macOS: first launch may show "cannot be opened because it is from an unidentified developer". Right-click the file → Open → Open. The script needs Python 3 (preinstalled on modern macOS) and will auto-install Ollama via Homebrew or the official installer.
  • Windows: double-clicking opens a console window. SmartScreen may show a "Windows protected your PC" warning — click "More info" → "Run anyway". Needs Python 3 from python.org or the Microsoft Store. Ollama is auto-installed via winget.
  • Linux: chmod +x once, then double-click or run from a terminal. Ollama is installed via the official `curl | sh` after a Y/N prompt.
